# SR-01-Commit-Discipline

## 📖 Deskripsi
Sub-Rak ini memfokuskan pada etika kerja profesional dalam menyusun sejarah Git. Fokus utamanya adalah memastikan sejarah kode tetap bersih, mudah dibaca, dan dapat dipertanggungjawabkan.

## 🏗️ Struktur Buku
1. **[BK-01-Atomic-Commits](./BK-01-Atomic-Commits/)**: Memahami komitmen fungsional tunggal.
2. **[BK-02-Conventional-Commits](./BK-02-Conventional-Commits/)**: Standarisasi format pesan commit.
