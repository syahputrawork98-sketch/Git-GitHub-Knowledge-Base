# 📊 BK-01: Capacity & Planning

> **"Beban kerja yang seimbang adalah kunci dari tim yang produktif dan bahagia."**

Buku ini membedah bagaimana menggunakan data GitHub Projects v2 untuk melakukan **Capacity Planning**. Kita akan belajar cara melihat beban kerja per anggota tim melalui grafik dan *Grouping*, serta cara menyesuaikan prioritas agar target rilis tetap realistis tanpa mengorbankan kesejahteraan tim (*Burnout Prevention*).
