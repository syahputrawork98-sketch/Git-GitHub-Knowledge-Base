# SR-01-History-Surgery

## 📖 Deskripsi
Sub-Rak ini memfokuskan pada pembersihan sejarah sebelum dipublikasikan, menggabungkan commit yang berantakan, dan menghapus jejak file sensitif dari histori.

## 🏗️ Struktur Buku
1. **[BK-01-Interactive-Rebase](./BK-01-Interactive-Rebase/)**: Seni merapikan commit secara interaktif.
2. **[BK-02-History-Sanitization](./BK-02-History-Sanitization/)**: Teknik menghapus data permanen dari histori.
