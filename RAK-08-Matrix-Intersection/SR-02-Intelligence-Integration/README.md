# SR-02-Intelligence-Integration

## 📖 Deskripsi
Sub-Rak ini mengeksplorasi bagaimana inteligensi buatan (AI) dapat meningkatkan efisiensi alur kerja kontrol versi dan bagaimana Git melacak konteks untuk AI.

## 🏗️ Struktur Buku
1. **[BK-01-AI-Workflow](./BK-01-AI-Workflow/)**: Kolaborasi AI-Human dalam ulasan kode dan ulasan komit.
2. **[BK-02-Context-Management](./BK-02-Context-Management/)**: Bagaimana AI memahami silsilah histori (DAG) untuk memberikan saran yang lebih baik.
